核心特点

  • MPPT 最大功率追踪，效率高达 99.9%
  • 支持同时满功率充电和放电
  • 支持三元 / 铁锂 / 铅酸 / 胶体多种电池
  • 支持锂电池、铅酸电池激活
  • 支持充电电流与充满设置
  • 支持温度补偿功能
  • 支持多种负载工作模式（手动 / 自动）
  • 支持保存历史数据
  • RS485 / TTL 标准 Modbus 通讯
  • 蓝牙 / 以太网通讯（选配外扩）
